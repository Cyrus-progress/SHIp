# SHiP — Silica-Histidine Polyplex platform

**An oxidation-proof delivery vehicle for ambient-temperature mRNA vaccines.**

This repository hosts the public website, computational model, and full
proposal materials for the SHiP project. Pre-seed proposal, April 2026.

The live website is at **[cyrussyho.github.io/ship-platform](https://cyrussyho.github.io/ship-platform)**

## Stage

The platform is at **pre-seed stage**. Computational evidence only;
no wet-lab validation has been performed yet. Numerical predictions
in this material are projections from kinetic modeling, not
measurements. See the website's "Key risks" section for an honest
assessment.
